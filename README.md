### Setup

Must have the latest npm version

* install:

```commandline
npm install 
```

* upgrade:

```upgrade
npm install -g npm@latest
```

### How to initialize application

```commandline
npm run dev
```

Go to: [http://localhost:3000/](http://localhost:3000/)
