### Setup

Must have the latest npm version

* install:

```commandline
npm install 
```

* upgrade:

```upgrade
npm install -g npm@latest
```

### How to initialize application

#### Deployed URL

[http://localhost:3000/](http://localhost:3000/)

#### Front End

```commandline
npm run dev
```

[http://localhost:3000/](http://localhost:3000/)
